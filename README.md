# DGVT TYPO3-Website

This project is the DGVT TYPO3-Websites.
It offers:

- DDEV configuration
- TYPO3 configuration
- Default Extensions
- Webpack Encore
- Linting & Testing
- Deployment (manual and automatic)
- customer_sitepackage (for the project specific customizations)
- etc.

You can find the [documentation here](docs/README.md).
